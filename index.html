Aprendendo sobre Git e GitHub

Comandos pra o Git

  • Git Init (Criar um repositorio)
  • Git Add . (O ponto serve para adicionar todos os arquivos no commit)
  • Git Commit -m 'Name-Commit' (Enviar primeiro commit)
  • Git Checkout -b 'New-Branch' (Criar uma nova branch, para criar o seu sem b.o para branch master)
  • Git Merge 'Name-branch' (Para fundir o seu projeto na branch master)
  • Git Status (Para saber o status do seus arquivos)
  • Git Log (Para saber seus commit e poder resgatar alguma Hex de commits anteriores)
  • Git Checkout 'HEX Branch' (Para entrar em commits anteriores sem perder nada)
  • git branch -D 'Name-branch' (Para apagar o branch localmente)
  • git push 'Name-Origin' 'Name-Branch' --delete (Para apagar o branch remotamente)

Comandos para o GitHub

  • Git Init (Cria um repositorio)
